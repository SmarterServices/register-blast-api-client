# register-blast-api-client
## Introduction


This lib is used as a wrapper around api calls to register blast api

## Installation

### Requirements

NODE v4.0 or higher

### Install Process
npm install

### Configuration
	var clientLib = require('register-blast-api-client')
	var client = new clientLib({url:'baseUrl',token:'auth token'})
